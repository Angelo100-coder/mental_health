# Mental Health Analysis

## Overview

This project explores the relationship between lifestyle factors and happiness using a structured dataset. The analysis focuses on variables such as sleep duration, exercise level, screen time, and social interaction.

## Objectives

- Analyze relationships between lifestyle variables and happiness
- Visualize trends using scatter plots and heatmaps
- Apply regression modeling to evaluate predictive relationships

## Data Processing

- Cleaned dataset and handled categorical variables using encoding
- Selected relevant numerical features for analysis
- Prepared data for visualization and modeling

## Exploratory Analysis

- Scatter plots were used to examine relationships between:
  - Sleep vs Happiness
  - Exercise vs Happiness
  - Screen Time vs Happiness
  - Social Interaction vs Happiness
- Correlation heatmap was generated to evaluate linear relationships

## Key Findings

- No strong correlations were found between lifestyle factors and happiness in this dataset
- Regression analysis showed low predictive power
- Data appears to lack strong underlying structure

## Tools Used

- Python
- Pandas
- Seaborn
- Matplotlib
- Statsmodels

## Conclusion

The analysis suggests that the dataset does not demonstrate meaningful relationships between lifestyle factors and happiness. This may indicate that the data is randomly generated or does not capture real-world behavioral patterns effectively.

## Limitations

-This dataset appear to lack strong underlying structure
-Results may be influenced by synthetic or randomly distributed data
-Real-world factors affecting happiness may not be fully captured

## Future Work

-Analyze a real-world dataset for comparison
-Incorporate additional variables such as income, health metrics, or demographics
-Apply more advanced modeling techniques
